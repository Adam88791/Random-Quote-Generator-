# Random-Quote-Generator

A simple and responsive Random Quote Generator built with HTML, CSS, and JavaScript. Get inspired with a new quote every time you click the button.

## Features
- Generates a random quote with a single click
- Responsive design for all devices
- Clean and simple UI

## How to Use
1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Click the "Generate Quote" button to see a new quote.

## Technologies
- HTML
- CSS
- JavaScript
